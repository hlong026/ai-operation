

## ❓ 常见问题

### Q: 端口被占用怎么办？
A: 修改 `vite.config.ts` 中的端口号：
```typescript
server: {
  port: 3001, // 改为其他端口
}
```

### Q: 如何连接后端？
A: 创建 `src/services/api.ts` 配置API地址：
```typescript
const API_BASE_URL = 'http://localhost:8000/api'
```

### Q: 数据是真实的吗？
A: 目前使用模拟数据，连接后端后会使用真实数据。
