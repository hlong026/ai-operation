## 🚀 如何使用

### 第一步：进入项目
```bash
cd ai-operation-frontend
```

### 第二步：安装依赖
```bash
npm install
```

### 第三步：启动项目
```bash
npm run dev
```

### 第四步：访问应用
打开浏览器访问：http://localhost:3000
