# AI自动运营系统 - 前端项目总结

## 项目概述

这是一个完整的AI自动运营系统前端项目，使用现代化的技术栈构建，包含13个功能页面，覆盖了从用户注册到内容创作、数据分析的完整流程。

## 技术栈

### 核心技术
- **React 18.3.1** - 最新的React版本
- **TypeScript 5.5.3** - 类型安全
- **Vite 5.4.1** - 极速构建工具

### UI框架
- **Ant Design 5.20.0** - 企业级UI组件库
- **@ant-design/icons 5.4.0** - 图标库

### 路由和状态
- **React Router 6.26.0** - 路由管理
- **Zustand 4.5.0** - 轻量级状态管理

### 工具库
- **Axios 1.7.0** - HTTP客户端
- **dayjs 1.11.10** - 日期处理

## 快速启动

### 方式1: 使用npm命令

```bash
cd ai-operation-frontend
npm install
npm run dev
```

### 方式2: 使用启动脚本

```bash
cd ai-operation-frontend
./start.sh
```

### 方式3: 手动启动

```bash
cd ai-operation-frontend
npm install
npm run dev
```

访问 http://localhost:3000 查看项目

## 页面路由

| 路由 | 页面 | 说明 |
|------|------|------|
| `/` | 首页 | 展示快速操作和推荐内容 |
| `/login` | 登录 | 用户登录 |
| `/register` | 注册 | 用户注册 |
| `/workflows` | 工作流列表 | 浏览所有工作流 |
| `/workflows/:id` | 工作流详情 | 查看工作流详细信息 |
| `/workflows/upload` | 上传工作流 | 上传新的工作流 |
| `/workflows/:id/execute` | 执行工作流 | 运行工作流 |
| `/tools` | 工具箱 | AI工具集合 |
| `/create` | 一键创作 | 内容创作流程 |
| `/analytics` | 数据分析 | 数据统计和分析 |
| `/accounts` | 账号管理 | 多平台账号管理 |
| `/team` | 团队管理 | MCN团队功能 |
| `/pricing` | 会员套餐 | 套餐选择和升级 |
