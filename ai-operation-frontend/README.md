# AI运营工具平台

基于 React + TypeScript + Ant Design + Supabase 构建的 AI 运营工具平台。

## 核心功能

### 🚀 工作流商店
- 浏览和使用各种 AI 工作流（Coze、n8n 等）
- 按分类和类型筛选
- 使用工作流消耗积分

### 🤖 智能体商店
- AI 智能助手，对话即服务
- 各类场景的智能体（内容创作、客服、翻译等）
- 使用智能体消耗积分

### 🔧 工具箱
- 各类实用 AI 工具集合
- 使用工具消耗积分

### ✏️ 创作者中心
- 上传工作流、智能体和工具
- 提交审核后发布
- 查看收益和提现

### 💰 积分系统
- 新用户注册赠送 100 积分
- 会员每月赠送积分
- 积分充值套餐
- 使用工作流/工具消耗积分
- 创作者获得 70% 分成
- 收益可提现

## 积分计费流程

```
用户获取积分:
├── 注册赠送 100 积分
├── 开通会员每月赠送积分
└── 直接充值购买积分

用户消费积分:
├── 使用工作流 → 扣除积分
└── 使用工具 → 扣除积分

创作者收益:
├── 用户使用时获得 70% 分成
├── 收益累计到可提现余额
└── 申请提现（最低 100 积分）

管理员设置:
├── 审核工作流/工具
├── 设置每个资源的积分价格
└── 设置分成比例
```

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 配置 Supabase

```bash
cp .env.example .env
# 编辑 .env 填入 Supabase 配置
```

### 3. 初始化数据库

在 Supabase SQL Editor 依次执行：
1. `supabase/schema.sql` - 基础表结构
2. `supabase/credits_system.sql` - 积分系统

### 4. 启动开发

```bash
npm run dev
```

## 数据库表

| 表名 | 说明 |
|------|------|
| `profiles` | 用户资料（含积分、收益） |
| `workflows` | 工作流（含积分价格、分成比例） |
| `agents` | 智能体（含积分价格、分成比例） |
| `tools` | 工具（含积分价格、分成比例） |
| `credit_packages` | 积分充值套餐 |
| `membership_plans` | 会员套餐 |
| `credit_transactions` | 积分交易记录 |
| `withdrawals` | 提现记录 |
| `orders` | 订单记录 |
