# AI运营系统

一个基于AI的智能内容运营平台，帮助创作者高效生成和管理内容。

## ✨ 主要功能

- 🚀 **一键创作** - 快速生成优质内容
- 📝 **AI选题** - 智能推荐热门话题
- ⚡ **工作流管理** - 自动化内容生产流程
- 📊 **数据分析** - 全面的运营数据洞察
- 🛠️ **工具箱** - 丰富的AI辅助工具

## 🎯 核心工具

- **文案提取工具** - 从视频中智能提取文案内容
- **文案二创工具** - 基于原始内容AI生成创新文案
- **账号拆解工具** - 深度分析目标账号运营策略
- **视频拆解工具** - 全方位分析视频内容结构

## 🚀 快速开始

### 安装依赖

```bash
cd ai-operation-frontend
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

## 🛠️ 技术栈

- **前端框架**: React 18 + TypeScript
- **UI组件库**: Ant Design 5
- **路由**: React Router 6
- **状态管理**: Zustand
- **构建工具**: Vite
- **HTTP客户端**: Axios
